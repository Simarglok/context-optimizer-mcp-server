# Code of Conduct

## Our Pledge
We are committed to providing a welcoming and inclusive environment for all contributors and users. Harassment, discrimination, or harmful behavior will not be tolerated.

## Our Standards
Examples of positive behavior:
- Respectful and constructive communication
- Being helpful and patient with others
- Accepting feedback gracefully
- Focusing on what is best for the community

Unacceptable behavior includes:
- Harassment, insults, or personal attacks
- Discriminatory language or imagery
- Trolling or inflammatory comments
- Publishing others' private information

## Enforcement
Report violations to the maintainers via GitHub issues (for general concerns) or privately via the repository owner’s contact listed in `package.json`.

Consequences may include warnings, temporary restrictions, or permanent bans from participation.

## Scope
This Code applies within all community spaces and in public when representing the project.
